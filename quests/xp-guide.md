# XP 획득 가이드

## 퀘스트 유형별 XP

| 타입 | 설명 | XP |
|------|------|----|
| 📖 읽기 | 기술 블로그 / 공식 문서 읽고 요약 | +5 XP |
| ✍️ 개념 정리 | 모르는 개념 노트 작성 | +10 XP |
| 💡 면접 Q&A | 예상 면접 질문 답변 정리 | +15 XP |
| 💻 실습 | 직접 코드 작성 및 실행 | +20 XP |
| 🔍 회사 분석 | 채용 공고 / 기술 블로그 분석 | +10 XP |
| 🏆 미니 프로젝트 | 기술 적용한 작은 프로젝트 완성 | +50 XP |

## 보너스 XP

| 조건 | 보너스 |
|------|--------|
| 3일 연속 학습 | +10 XP |
| 7일 연속 학습 | +30 XP |
| 30일 연속 학습 | +100 XP |
| 하루 3개 이상 퀘스트 완료 | +5 XP |
| 하루 5개 이상 퀘스트 완료 | +15 XP |
