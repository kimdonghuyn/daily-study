# Daily Study

매일 아침 목표 회사들의 기술 스택을 분석하고 학습하는 프로젝트.

## 구조

```
daily-study/
├── companies/        # 회사별 기술 스택 분석
│   └── [회사명]/
│       ├── overview.md      # 회사 개요 & 채용 요건
│       ├── tech-stack.md    # 기술 스택 정리
│       └── study-notes.md   # 학습 내용 메모
├── daily-log/        # 날짜별 학습 일지
│   └── YYYY-MM-DD.md
└── resources/        # 공통 참고 자료
```

## 진행 방식

1. 목표 회사 선정 → `companies/` 아래에 폴더 생성
2. 채용 공고 / 기술 블로그 분석
3. 부족한 부분 파악 → 학습
4. `daily-log/` 에 그날 배운 것 기록
