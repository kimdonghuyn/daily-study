## 진행 방식

1. 저녁 — Slack에 `!학습 [오늘 배운 내용]` 입력 → 내일 아침 관련 토픽 질문 발송
2. 08:00 — AI가 오늘의 유형(개념/문제/면접 순환) 질문 1개 발송 (사이클별 난이도 자동 조절)
3. 낮 — 채널에 답변 입력 → 즉시 피드백 + 정확도 채점 (0~100점)
4. 13:00 — 모범 답안 공개 + XP 적립 + 학습 기록 GitHub 자동 커밋
